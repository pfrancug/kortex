# @nexgraph/core

GPU-first **WebGL2** graph engine for the browser: renderer, typed graph buffers, worker force layout, parsers, picking helpers.

Consumes **typed arrays**; no React in this package. React bindings live in [**`@nexgraph/react`**](https://www.npmjs.com/package/@nexgraph/react).

```ts
import { Renderer, ForceLayout, parseGraphAsync } from '@nexgraph/core';
```

See the monorepo README for demos and integration guides.
