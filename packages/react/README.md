# @nexgraph/react

Thin React bindings over **`@nexgraph/core`** — mounts a **`Renderer`**, declarative **`dataset`** / **`graph`** props, optional **`ForceLayout`**, picking callbacks.

```tsx
import { NexgraphCanvas } from '@nexgraph/react';

<NexgraphCanvas dataset={…} />
```

Requires **`react`** and **`react-dom`** as peers. Engine: [**`@nexgraph/core`**](https://www.npmjs.com/package/@nexgraph/core).
